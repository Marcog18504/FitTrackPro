# FitTrackPro

FitTrackPro e' un'app mobile sviluppata con React Native ed Expo per gestire esercizi, schede di allenamento, sessioni pianificate, allenamenti svolti e obiettivi personali.

## Requisiti

- Node.js installato
- npm installato
- Expo Go installata sul telefono, se si vuole provare l'app su dispositivo fisico

## Installazione

Dalla cartella del progetto eseguire:

```bash
npm install
```

## Avvio dell'app

Per avviare Expo:

```bash
npm start
```

Dopo l'avvio si puo':

- scansionare il QR code con Expo Go per aprire l'app su telefono;
- premere `w` nel terminale per aprire la versione web;
- usare gli script dedicati:

```bash
npm run web
npm run android
npm run ios
```

## Controllo TypeScript

Per verificare che il codice TypeScript non contenga errori:

```bash
npx tsc --noEmit
```
